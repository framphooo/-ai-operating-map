# Contributing

Thanks for your interest in improving AI Operating Map. Here is how to get set up.

## Prerequisites
- Node.js 18+
- npm (comes with Node)

## Local setup
```bash
npm install
npm run dev
```
Then open http://localhost:3000.

## Quality checks
- `npm run lint` before opening a PR.

## Submitting changes
- Keep PRs focused and include a brief description of the change and why.
- If you add configuration keys, document them in `env.example` and avoid committing secrets.

## Reporting issues
- Open a GitHub issue with steps to reproduce and expected/actual behavior.
